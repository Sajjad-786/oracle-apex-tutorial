# Oracle APEX Tutorial

This repository documents a complete, step-by-step Oracle APEX tutorial.

## Scope
- Database schema design (Companies, Users, etc.)
- Authentication & authorization concepts
- Oracle APEX application structure
- Best practices & enterprise patterns

## Structure
- `database/ddl` – Database foundation scripts
- `apex` – APEX pages, auth, security
- `docs` – Written tutorial chapters

This project evolves incrementally.
